Displays ETH and BTC tickers in ubuntu dekstop bar.

Tested on ubuntu 16.04

##Setup
1. Clone repo

## Configure start on boot
1. chmod +x startup-eth.sh && chmod +x startup-btc.sh
2. add startup-btc.sh & startup-eth.sh to startup programs
